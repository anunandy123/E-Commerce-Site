# E-Commerce-Site
## Project Description

This project is a Django-based e-commerce web application developed for managing and purchasing fashion products online. It provides a user-friendly interface where customers can browse products by category, view detailed product information, register and log in, manage their shopping cart, and proceed to checkout.

The application includes product management features such as categories, pricing, descriptions, images, sizes, colors, brands, and customer reviews. It demonstrates the use of Django's MVC architecture, database integration, template rendering, and media file handling to build a functional online shopping platform.

This project was developed as a practical implementation of full-stack web development concepts using Python and Django.

## Features

- User Registration and Login
- Product Categories
- Product Details Page
- Shopping Cart
- Checkout Page
- Product Images
- Size and Color Selection
- Customer Reviews
- Responsive User Interface
- Django Admin Panel for Data Management

## Technologies Used

- Python
- Django
- HTML5
- CSS3
- SQLite
- Bootstrap
